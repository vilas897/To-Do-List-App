# To do list App

A Basic To-Do List App that can add and delete items. Items are saved in a database for future retrieval

Tools used:
- MongoDB 
- ExpressJS
- NodeJS
- HTML CSS
- Ajax
<br><br>

## Installing Node.js

Follow the instructions to install node and npm.

https://nodejs.org/

## Installing dependencies

`npm install`

## Runnning the server

`npm start`
